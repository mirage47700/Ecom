⚙️ Configuration des API Keys

🤖 Gemini API (Google AI)

  1. Aller sur aistudio.google.com/apikey
  2. Créer une clé API gratuite
  3. Coller dans le nœud ⚙️ Config
geminiApiKey: "AIza..."

✅ Gratuit — 1 500 req/jour sur Gemini 2.0 Flash

🔍 SerpAPI

  1. Aller sur serpapi.com
  2. Créer un compte (100 req/mois gratuits)
  3. Copier la clé API depuis le dashboard
  4. Coller dans le nœud ⚙️ Config
serpApiKey: "abc123..."

⚠️ Google Trends + Search = 2 req par produit

📊 Google Sheets (OAuth2)

  1. Dans n8n → Credentials → Add → Google Sheets OAuth2
  2. Autoriser l'accès à ton compte Google
  3. Créer un Google Sheet avec une feuille "Recherches"
  4. Copier l'ID du sheet depuis l'URL
Sheet ID: "1BxiMVs0..." (dans l'URL)

🎯 Niche à analyser

Changer le champ niche dans le nœud ⚙️ Config

niche: "home decor"

Exemples :

outdoor furniture pet accessories kitchen gadgets baby products fitness equipment
🤖 Prompt Gemini (Step 1)

Ce prompt est envoyé automatiquement à Gemini 2.0 Flash avec le paramètre niche :

Tu es un expert en e-commerce dropshipping spécialisé en product research pour Google Ads. Niche à analyser: {{ niche }} Génère exactement 10 produits evergreen data-driven. CRITÈRES OBLIGATOIRES: - Produits physiques (non digitaux, non alimentaires) - Demande stable ou croissante depuis minimum 5 ans - Forte intention d'achat (buyer intent keywords) - Volume de recherche estimé > 10 000 / mois sur Google - Facilement sourcable sur AliExpress/Alibaba - Aucune marque déposée (produits génériques uniquement) POUR CHAQUE PRODUIT: { "name": "nom en anglais", "category": "sous-catégorie", "main_keyword": "keyword principal", "keywords": ["kw1", "kw2", "kw3", "kw4", "kw5"], "buyer_intent": "description courte", "competition": "low|medium|high", "aliexpress_query": "terme de recherche AliExpress", "evergreen_reason": "explication courte" } Réponds UNIQUEMENT avec du JSON valide (responseMimeType: application/json)
