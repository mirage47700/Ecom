# Ecom Research Tracker

Outil de recherche produit e-commerce — **Buyers → Intention → Google ADS**

## Process en 7 étapes

| Étape | Action |
|-------|--------|
| **1** | ChatGPT — Trouver des idées de keywords (Keyword Planner) |
| **2** | Prompt — Affiner la recherche avec un prompt structuré |
| **3-4** | Google Trends — Analyser la courbe sur 5 ans / 12 mois / 90 jours |
| **5** | Google Keyword Planner — 5 keywords principaux + CPC + searches/mo |
| **6** | Concurrents — `site:myshopify.com [produit]` + `/collections/all?sort_by=best-selling` |
| **7** | Décision — GO / NO-GO / En attente |

## Objectifs Revenue

| Produits | Revenue/mois |
|----------|-------------|
| 750+     | 100k$       |
| 1 500+   | 200k$       |
| 2 000+   | 300k$       |
| 2 500+   | 400k$       |

## Utilisation

Ouvrir `index.html` dans un navigateur. Les données sont sauvegardées localement (localStorage).
